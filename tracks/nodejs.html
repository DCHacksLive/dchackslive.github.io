Node.js

Node.js is the server-side portion of the term "full-stack JavaScript." Node.js is completely asynchronouse and allows for features like easy URL routing and database access to create dynamic, contextual, and responsive web applications. For the modern web, Node.js has emerged to be the weapon of choice.

Scotch.io

Scotch.io has many great project walkthroughs, but its Node.js and MEAN stack tutorials are some of its most famous. They may not tell you about everything going on under-the-hood, and your understanding may still be a little sketchy after going through one of their tutorials, but Scotch.io is still an excellent place to start getting acquainted with Node.js.

Learn All the Nodes

Learn All the Nodes is great if you want to really understand everything that's happening under-the-hood with Node.js, but even if you don't, its beginner videos are enough to help you get your app successfully running.

Node Tuts

If you're already somewhat of a Node.js aficionado and just want to learn more about the #OneTrueProgrammingLanguage, then Node Tuts might offer what you want. They have several short, internet-accessible books written on different aspects of Node.js, exploring in-depth the language. Rather than being a full-blown tutorial, Node Tuts fulfills the role of being the perfect quick reference when making a project of your own.

Chrome DevTools

I'll be leaving this at the end of every Web Development track because, for the love of God, use this as much as you can. It can be used to debug some simple HTML and CSS or a full-stack web application. No matter your expertise, DevTools is your friend.
